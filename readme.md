## dots

Config files for my first Linux machine, a revived ThinkPad X1 Carbon.

For the last 12 years all my devices have followed a Pokemon naming scheme. This one is called `rotom`.

### choices

- os: arch linux
- window manager: hyprland
- terminal: ghostty
- status bar: waybar
- editor: neovim
- browser: brave
- wallpaper: hyprpaper
- screenshot: hyprshot
